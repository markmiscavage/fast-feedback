Sbk Feedback
===========================================================

Django admin login
-----------------------------------------------------------

* Username: admin
* Password: notinproduction

Installing Sbk Feedback Backend
-----------------------------------------------------------

```sh
cd <PATH_TO_PROJECT>
sh ./scripts/setup.sh
```

Installing Sbk Feedback Frontend
-----------------------------------------------------------

```sh
cd <PATH_TO_PROJECT>
npm install
```

Running the project
-----------------------------------------------------------

```sh
cd <PATH_TO_PROJECT>
grunt server:8001
```
